Windows Powershell Özelleştirme Nasıl Yapılır?

Oluşturulma Tarihi29-09-2023|Güncellenme Tarihi29-09-2023|Windows
|Görüntülenme Sayısı:

Windows Terminal kurulumunu Microsoft Store üzerinden yapalım.

Terminalimiz aşağıdaki gibi görünecektir.

“Aşağı ok” işaretine tıkladıktan sonra “Settings” seçeneğine tıklayarak ayarlara gidelim.

Sol alt köşedeki “Open JSON file” seçeneğine tıklayalım.

Buradaki bağlantıda yer alan font’ları indirip bilgisayarımıza kuralım. Siz daha farklı font’lar da kullanabilirsiniz. Bilgisayarımıza kurmuş olduğumuz Font’u seçelim.

Renk şemasını seçeceğiz. Buradaki bağlantıda kullanabileceğimiz renk şemalarını görebilirsiniz. Beğendiğiniz bir renk şemasını aşağıdaki biçimde dosyamıza ekleyelim.

Eklemiş olduğumuz renk şemasını seçelim.

Herhangi bir profil oluşturulup oluşturulmadığını kontrol etmek için aşağıdaki komutu girelim.

Profil oluşturulmadıysa yukarıdaki gibi “False” değeri dönecektir. False değeri döndüğü için aşağıdaki komutu kullanarak yeni bir profil oluşturalım.

Oh-my-posh kurulumu yapabilmek adına scoop kurulumunu yapalım. Terminal üzerinde aşağıdaki komutu girelim.

Execution-Policy ile alakalı bir hata alırsanız aşağıdaki komutu giriniz.

Şimdi scoop ile oh-my-posh kurulumunu gerçekleştirelim.

Kurulumu gerçekleştirdikten sonra aşağıdaki komut ile profilimizi açalım.

Kaydettikten sonra Terminal’i tekrar açalım ve görünümümüze bakalım.

Buradaki bağlantı üzerinden bir tane tema seçelim. İlgili json dosyasını bilgisayarımızda uygun bir yere kaydettikten sonra profil dosyamızı şu şekilde düzenleyelim.

Terminal görünümü şu şekilde değişecektir.
